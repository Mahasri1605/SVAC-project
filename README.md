# Inventory Management Analysis using Data Insights

## Project Overview
This project is a web-based Inventory Management Analysis System developed using Flask, HTML, CSS, and JavaScript. The system helps in managing inventory data and generating useful insights for better business decision-making.

## Features
- Inventory stock tracking
- Product category management
- Low stock detection
- Out of stock analysis
- Inventory value calculation
- Sales and stock insights
- Interactive dashboard and charts

## Technologies Used
- Python Flask
- HTML
- CSS
- JavaScript

## Project Structure
- app.py – Backend Flask application
- templates/ – HTML frontend files
- static/ – CSS and JavaScript files
- requirements.txt – Required Python packages

## How to Run
1. Install required packages:
pip install -r requirements.txt

2. Run the Flask application:
python app.py

3. Open browser and visit:
http://127.0.0.1:5000/

## Conclusion
The project provides an efficient way to analyze inventory data using data insights techniques. It helps businesses monitor stock levels, identify shortages, and improve inventory management performance.
